# matMul
Каждый поток видеокарты получает задачу вычислить ровно один элемент матрицы C. Поскольку размер матрицы C составляет 1024×768 = 786 432 элемента, запускается ровно столько же потоков

Каждый элемент C[row][col] вычисляется на основе: строки матрицы A и столбца матрицы B.
При этом результат для C[0][0] никак не зависит от результата для C[0][1] или любого другого элемента. Это значит, что все 786 432 элемента можно вычислять одновременно и независимо

Вся задача разбивается на блоки, каждый блок — на потоки. Параметр BLOCK_SIZE = 16 означает, что один блок содержит 16×16 = 256 потоков. Количество блоков вычисляется так, чтобы покрыть всю матрицу C.

Каждый поток определяет свои глобальные координаты: row = by * 16 + ty (номер строки в C) и col = bx * 16 + tx (номер столбца в C). Затем поток выполняет скалярное произведение соответствующей строки из матрицы A и столбца из матрицы B, записывая результат в C[row][col].
